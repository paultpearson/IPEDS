PPBACC - Number of Postbaccalaureate certificate programs offered
Number of Postbaccalauureate certificate programs is derived from the total Postbaccalureate certificate awarded, which are reported on the completions component for the 12-month period July 1 - June 30. If the total Postbaccalaureate certificates were greater than or equal to zero for each 6-digit CIPCODE (program), by award level, the number of Postbaccalaureate certificate programs offered via distance education was set to 1 - program offered, else program offered was set to zero – not offered. The 2-digit CIPCODE level contains the total number of 6-digit programs offered for the 2-digit category and the grand total CIPCODE (99) will contain the number of programs for the institution. (Note: Institutions are asked to enter a zero to indicate a program that is offered, but no awards were granted in the 12-month period). Values and labels 1 - program offered, applicable for 6-digit cipcodes 0 - Program not offered, applicable for 6-digit cipcodes 1-High - Total programs offered, applicable for 2-digit and 99 grand total summary records 0 - No programs offered, applicable for 2-digit and 99 grand total summary records
